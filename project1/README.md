# Your First Project
如果你已經熟悉HTML/CSS/Angular Material，可以跳過此練習～  

## Goal
建立一個簡單的學生註冊單，需有下面的功能：  

1. 欄位項目與格式  
2. 送出欄位之後顯示結果  

```
    {
        firstName: STRING,
        lastName: STRING,
        gender: STRING, (select from M/F),
        birthday: DATE,
        phones: [
            NUMBER, NUMBER...
        ],
        addresses: [
            {
                street: STRING,
                city: STRING,
                state: STRING, (select)
                zipCode: NUMBER
            },
            ...
        ]
    }
```

## Approach
1. HTML5 / CSS  
    可使用任何library, Angular Material, Bootstrap...  
2. Javascript  
    可使用純Javascript或是任何library如jQuery等...  

## Note
不需要做得太花俏，有功能可以使用即可。
